Sound

Sound:

AppleWin emulates the Apple II's internal speaker, the Mockingboard (or Phasor) sound card and also the SAM card.

Use the AppleWin Configuration tab labelled Slots to select sound card(s) and configure them.

Speaker:

You can now only emulate this with a PC sound card.

Mockingboard:

A 'Mockingboard C' card is a 'Mockingboard A' card with an SSI263 speech chip (typically the older SSI263P).

Mockingboards can be plugged into any Apple II slot.

Each Mockingboard is a type 'C': 2x AY-3-8913 (3-channel sound generator) and an SSI263 (speech chip)
Each AY-3-8913 is fully emulated (tone, noise and envelope).
Each SY6522 (that controls an AY-3-8913) can generate precise timer interrupts or phoneme-complete interrupts.

NB. Some Mockingboard compatible software requires the card to be in a specific slot (usually slot 4).
Some software support dual Mockingboards in slots 4 and 5 (eg. Ultima V and a few French Touch demos).
Some software will allow the user to pick the slot, and better software will auto-detect the slot.

Phasor:

A Phasor card includes an SSI263 speech chip (typically the newer SSI263AP).

Phasors can be plugged into any Apple II slot.

A Phasor is roughly equivalent to 2 Mockingboards and has 4x AY-3-8913.
It powers up in a hardware-identical mode to the Mockingboard type 'C', and so is compatible with all Mockingboard titles.
Using software (to access memory mapped I/O) it can be switched to native Phasor mode (4x AY8913s) or Echo+ mode (but without the TMS5220 speech chip).

Speech chips:

Both the Mockingboard C and Phasor cards support up to two SSI263s and an aftermarket hardware modification for a Votrax SC-01 (the predecessor to the SSI263).
Emulation of the SSI263 is basic at the moment and only plays back phonemes at a fixed rate without any support for inflection or filters.
Two types of SSI263 are supported: SSI263P (original, with a reset bug) and SSI263AP (updated with the reset bug fixed).
Additionally the Votrax SC-01 is simply emulated by remapping SC-01 phonemes to the SSI263's.

SAM:

SAM (Software Automatic Mouth) is an 8-bit DAC card than can be plugged into any Apple II slot.
