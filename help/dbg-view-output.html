View Output

Video Modes

You can view the output of the Apple by using the following commands.

Command

Effect

\

Display Apple output until a key pressed.

DGR

View Double lo-res (current page).

DGR1

View Double lo-res Page 1.

DGR2

View Double lo-res Page 2.

DHGR

View Double Hi-res (current page).

DHGR1

View Double Hi-res Page 1.

DHGR2

View Double Hi-res Page 2.

GR

View Lo-Res screen (current page).

GR1

View Lo-Res screen Page 1.

GR2

View Lo-Res screen Page 2.

HGR

View Hi-res (current page).

HGR0

View pseudo Hi-res Page 0 ($0000).

HGR1

View Hi-res Page 1 ($2000).

HGR2

View Hi-res Page 2 ($4000).

HGR3

View pseudo Hi-res Page 3 ($6000).

HGR4

View pseudo Hi-res Page 4 ($8000).

HGR5

View pseudo Hi-res Page 5 ($A000).

HGR6

View pseudo Hi-res Page 6 LC Banks 1/2 ($C000-$DFFF).

HGR7

View pseudo Hi-res Page 7 LC Banks 2/RAM ($D000-$EFFF).

HGR8

View pseudo Hi-res Page 8 LC Banks RAM ($D000-$EFFF).

SHR

View Super Hi-res.

TEXT

View Text screen (current page).

TEXT1

View Text screen Page 1 ($0400).

TEXT2

View Text screen Page 2 ($0800).

TEXT80

View 80-column Text screen (current page).

TEXT81

View 80-column Text screen Page 1.

TEXT82

View 80-column Text screen Page 2.

Notes:

  • If you change to a specific video mode/page different from the current video settings and then exit the debugger (F7) the screen will NOT change to the current video settings. This lets you inspect other video modes/pages until code (re)sets the video mode or page -- handy if reverse engineering games.

  • If the Apple's output is in a mixed mode then viewing a specific mode MAY view them in full screen mode.


Output Keys

While viewing the Apple's output you can use the following keys to switch video pages. Pressing any other key will return to the debugger's output.

Key

Name

Effect

0

Zero

View pseudo page 0
($0000-$1FFF) if graphics,
page 1 if text.

1

One

View page 1
($2000-$3FFF) if graphics,
($0400-$07FF) if text.

2

Two

View page 2
($4000-$5FFF) if graphics,
($0800-$0BFF) if text.

3

Three

View pseudo page 3
($6000-$7FFF) if graphics,
page 1 if text.

4

Four

View pseudo page 4
($8000-$9FFF) if graphics,
page 1 if text.

5

Five

View pseudo page 5
($A000-$BFFF) if graphics,
page 1 if text.

6

Six

View pseudo page 6
LC Banks 1/2 ($C000-$DFFF) if graphics,
page 1 if text.

7

Seven

View pseudo page 7
LC Banks 2/RAM ($D000-$EFFF) if graphics,
page 1 if text.

8

Eight

View pseudo page 8
LC Bank RAM ($E000-$FFFF) if graphics,
page 1 if text.

9

Nine

View current video mode and page.
